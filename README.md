# CLI.cr
Yet another Crystal command line interface framework!

## Installation
1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  cli:
    github: devnote-dev/cli.cr
```

2. Run `shards install`

## Usage
```crystal
require "cli"
```
